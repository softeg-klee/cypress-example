# INSTALLATION
* At the root of your project, create new directory (ex e2e)
```bash
mkdir e2e
cd e2e
```
* Install the cypress-example tool with npm or yarn.
```bash
yarn add {repo}
```
* Set up your project and follow the wizzard.
```bash
yarn set-up
```
